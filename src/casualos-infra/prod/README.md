## CasualOS Production Modules

OpenTofu modules that enable production deployments of CasualOS.

There are two types of modules:

-   bootstrap
    -   Bootstrap modules are in charge of setting up the cluster
-   deploy

There is currently one kind of OpenTofu production deployments:

-   AWS

If you are looking for a serverless deployment of CasualOS, use the CloudFormation template available [here](../../aux-server/aux-backend/serverless/aws/template.yml).
