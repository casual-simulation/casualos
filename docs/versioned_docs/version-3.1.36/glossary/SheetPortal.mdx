The sheetPortal is a <GlossaryRef term="portal">portal</GlossaryRef> that makes it easy to view and edit bots which are in a <GlossaryRef term="dimension">dimension</GlossaryRef>.

It can be opened by setting the <TagLink tag='sheetPortal'/> tag on the <VariableLink name="configBot"/> to a dimension (such as "home"), or by clicking <GlossaryRef term="ab-1"/> and selecting the "sheetPortal" button.

![Image of bots in the sheetPortal](/img/glossary/sheetPortal_01.png)

The sheetPortal organizes bots in columns where each bot is a column and each tag is represented along a row. Visually, this is similar to how a spreadsheet is displayed.
